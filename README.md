# Flower Shop Homepage UI

**Web Development Internship Task**

Hello! Thank you for the opportunity to complete this practical task. 

This repository contains my submission for the Web Development Internship Homepage UI Coding assignment. I have converted the provided visual guide into a clean, modern, and fully responsive web page.

## 🛠️ Tech Stack & Approach

For this assignment, I chose to use **HTML5** and **Tailwind CSS**. 

Since the role emphasizes strong fundamental web development skills and well-structured, logical code, I focused on writing this page manually with clean, semantic markup.

* **HTML5:** I used proper semantic tags (`<nav>`, `<section>`, `<footer>`, etc.) to ensure good document structure, accessibility, and SEO best practices.
* **Tailwind CSS:** I utilized Tailwind via CDN for styling. Instead of relying on heavy pre-built templates, I used Tailwind's utility classes to build custom Flexbox and Grid layouts from scratch. This allowed me to match the visual guide while keeping full control over the responsiveness.
* **No Heavy Frameworks:** To demonstrate my core frontend fundamentals, I avoided using JavaScript frameworks like React or Vue. The page relies entirely on HTML and CSS.

## 🚀 Features & Highlights

* **Fully Responsive:** The layout seamlessly adapts to Mobile, Tablet, and Desktop screens. I paid special attention to spacing and image scaling across different breakpoints.
* **Clean Code Structure:** The `index.html` file is organized logically. I added clear comments separating each section (e.g., Hero Section, New Products, Testimonials) to make the code easy for another developer to read and maintain.
* **Modern Typography:** Integrated Google Fonts ('Inter' for clean reading and 'Playfair Display' for elegant headings) to give the flower shop a premium feel.
* **Local Assets:** All images are saved locally in the `images/` directory to ensure fast, reliable loading without broken links.

## 💻 How to View the Project

Since this is a static frontend project, viewing it is very straightforward:

1. Extract the submitted ZIP folder.
2. Open the `index.html` file directly in any modern web browser (Chrome, Firefox, Safari, Edge).
3. *(Optional)* If you are reviewing the code in VS Code, you can use the "Live Server" extension for a live preview.
